Formularios: Todos los Tipos de Input y Atributos

<form> con atributos:

  • action="url" - Destino donde se envían los datos
  • method="post" - Método de envío (post: seguro, get: visible en URL)
  • autocomplete="on" - Activa el autocompletado del navegador
<legend> - Información Personal

Campos de Texto

Atributos: type="text", id="nombre", name="nombre", placeholder, required, autofocus

type="email" - Valida formato de correo electrónico

type="password" - Oculta el texto | minlength="8" - Mínimo 8 caracteres

type="tel" - Campo de teléfono | pattern - Expresión regular de validación

type="number" | min="18" max="100" step="1" - Rango y saltos numéricos

type="range" - Control deslizante (slider) | value="5" - Valor inicial

Fechas y Tiempos

type="date" - Selector de fecha | min/max - Rango de fechas válidas

type="time" - Selector de hora (sin zona horaria)

type="datetime-local" - Selector de fecha y hora combinados

type="month" - Selector de mes y año

type="week" - Selector de semana y año

Selecciones y Opciones

type="color" - Selector de color | value="#ff6347" - Color inicial

Radio Buttons (una opción)

type="radio" - Botones de opción única (mismo name)



Checkboxes (múltiples opciones)

type="checkbox" - Casillas de verificación múltiple



<select> - Lista Desplegable

size="1" - Muestra 1 opción | <option value selected> - Opción preseleccionada

size="4" - Muestra 4 opciones | multiple - Permite selección múltiple (Ctrl+Click)

<datalist> - Autocompletado

list="id" vincula input con datalist | Permite escribir o seleccionar

Archivos, URLs y Búsquedas

type="file" - Selector de archivos | multiple - Permite múltiples archivos

type="url" - Valida formato de URL

type="search" - Campo optimizado para búsquedas

<textarea> y Campos Especiales

rows="5" cols="50" - Define tamaño visible (5 filas x 50 columnas)

readonly - Campo visible pero no editable

disabled - Campo deshabilitado (no se envía en el formulario)

type="hidden" - Campo oculto (envía datos sin mostrarlos): usuario-id="12345"

Botones de Acción

type="submit" - Envía el formulario al servidor

type="reset" - Restablece todos los campos a sus valores predeterminados

<button type="button"> - Botón personalizable con JavaScript

type="image" - Botón de envío con imagen personalizada
